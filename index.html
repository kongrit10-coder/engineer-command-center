ยินดีต้อนรับ! ก่อนเริ่มใช้งาน กรุณาเชื่อมต่อ Google Sheets เพื่อเก็บข้อมูล

1

คัดลอก Google Apps Script

เปิด script.google.com → สร้างโปรเจกต์ใหม่ → วางโค้ดจากไฟล์ Code.gs → กด Deploy เป็น Web App (Anyone)

2

วาง Web App URL

หลัง Deploy จะได้ URL เช่น https://script.google.com/macros/s/xxx/exec

3

กด Connect แล้วเริ่มใช้งาน

ข้อมูลจะถูกเก็บใน Google Sheets ของคุณอัตโนมัติ
